# EDA Mini-Project - Netflix User Watching Behavior 

My first data analytics mini-project. Exploratory data analysis of a Netflix user behavior dataset from Kaggle, focused on finding patterns in subscriptions, genres, age groups, and demographics.

---

## Dataset

[Netflix User Watching Behavior Dataset](https://www.kaggle.com/datasets/rhythmghai/netflix-user-watching-behavior-dataset) by rhythmghai on Kaggle.

> Note: This is a synthetic dataset, meaning some distributions (gender) are artificially balanced and may not reflect real-world Netflix data.

---

## What I Analyzed

- **Subscription types** — which plan users prefer the most
- **Subscriptions by country** — geographic breakdown of subscription types
- **Favorite genres** — most popular genres among users
- **Age groups** — pie chart breakdown of user age distribution
- **Gender distribution** — how evenly split the user base is

---

## Key Insights

1. **Standard** is the most popular subscription type — users prefer the middle option between Basic and Premium
2. **Brazil** has the highest number of users in this dataset
3. **Documentary** is the most watched genre
4. Users aged **36–50** make up the largest age group
5. Gender distribution is nearly equal across Female, Male, and Other
